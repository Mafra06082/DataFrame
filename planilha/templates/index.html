TUTORIAL

Verifique os campos:

Na página principal, você já verá três campos de upload destinados às suas planilhas. Confira se os arquivos corretos estão prontos para serem enviados.

Selecione os arquivos:

Clique em cada campo de arquivo e escolha a planilha desejada no seu computador. Certifique-se de que os formatos dos arquivos estão conforme especificado (por exemplo, CSV ou XLSX).

Revise as instruções:

As orientações detalhadas estão à direita dos campos. Leia rapidamente para confirmar o formato e a sequência de carregamento, se necessário.

Clique em "GERAR":

Após inserir todos os arquivos, basta clicar no botão "GERAR" para iniciar o processo de compilação. O sistema combinará e organizará os dados automaticamente.

Aguarde a conclusão:

Enquanto o site processa os dados, você pode acompanhar o status (geralmente indicado por uma barra de progresso ou mensagem).

Resultado:

Assim que o processamento for concluído, uma nova planilha será gerada. Você poderá visualizá-la diretamente no site ou fazer o download.
